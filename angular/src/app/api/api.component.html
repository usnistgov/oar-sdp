This NIST Data Discovery web portal hosts access to a RestFul web service public API, Application Programming Interface. This API provides our developer community the capability to dynamically query the underlying data resource catalog, a collection of key NIST research dataset descriptions (metadata).

The search API returns JSON open-standard format data objects containing high-level resource information and scientific domain-specific metadata representations.
